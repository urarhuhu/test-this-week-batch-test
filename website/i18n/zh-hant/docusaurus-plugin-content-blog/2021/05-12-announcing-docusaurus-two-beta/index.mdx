## 為什麼 Docusaurus v2 的 alpha 階段持續這麼久？

很難相信第一個 alpha 版本 [v2.0.0-alpha.0](https://github.com/facebook/docusaurus/releases/tag/v2.0.0-alpha.0) 已經是 2 年前的事了 😳，這對軟體 alpha 階段來說異常漫長。

由於這是對程式碼庫的重大重新架構，我們希望在進入測試階段之前確保對 Docusaurus 2 的穩定性和功能有信心，因為測試版意味著高品質。我們很高興地宣布，Docusaurus 2 已通過 **[i18n](https://docusaurus.io/blog/2021/03/09/releasing-docusaurus-i18n)** 實現與 Docusaurus 1 的**功能對等**，並已被許多 Docusaurus 網站**成功採用**（參見 [i18n 展示](https://docusaurus.io/showcase?tags=i18n)）。

我們現在相當有信心 Docusaurus 2 的核心功能已足夠穩定，可以進入測試階段。

## 測試版的目標是什麼？

現在 Docusaurus 2 已穩定且主要功能完整，測試版的目標是讓新用戶對 Docusaurus 2 的生產就緒性建立信心，將更多仍在使用 Docusaurus 1 的用戶遷移到版本 2，並正式棄用 Docusaurus 1。當然，我們將繼續解決可能發現的任何問題和錯誤。

此外，我們將利用測試階段**改進我們的主題系統**。

我們希望使其：

- **更容易實現自訂主題**，包括對我們自己。我們希望長期提供[多個官方主題](https://github.com/facebook/docusaurus/issues/3522)（包括 [Tailwind CSS](https://github.com/facebook/docusaurus/issues/2961)）。

- **更安全地擴展現有主題**：升級高度自訂的 Docusaurus 網站可能會很痛苦，因為自訂可能與內部變更衝突。我們需要使主題的公共 API 表面更加明確，並清楚說明哪些是安全可自訂的。

我們將建立更好的**主題基礎設施**，並重構經典主題以使用它。

如果您自訂網站，可能會發現這些計劃中的改進相當有價值。

## 有什麼新功能？

如果您錯過了，我們最近推出了兩項重大改進：

- [自動生成的側邊欄](https://docusaurus.io/docs/sidebar#sidebar-item-autogenerated)：不再需要維護 `sidebars.js` 檔案！
- [Webpack 5 / PostCSS 8](https://github.com/facebook/docusaurus/issues/4027)：持久性快顯記憶體顯著加快了**重建時間**！

## 下一步是什麼？

推出官方 2.0 版本！

為了達成這一目標，我們將繼續**修復錯誤**並實現**最受期待的功能**，包括：

- [側邊欄分類索引頁面](https://github.com/facebook/docusaurus/issues/2643)
- [更好的行動導航使用者體驗](https://github.com/facebook/docusaurus/issues/2220)
- [更好的警示設計](https://github.com/facebookincubator/infima/issues/55)
- [CSS-in-JS 支援](https://github.com/facebook/docusaurus/issues/3236)
- [提升建置時間效能](https://github.com/facebook/docusaurus/issues/4765)
- [擴展 Docusaurus 插件，CMS 整合](https://github.com/facebook/docusaurus/issues/4138)
- [修正尾部斜線和相對連結問題](https://github.com/facebook/docusaurus/issues/3372)
- [更好的 CommonMark 相容性](https://github.com/facebook/docusaurus/issues/3018)
- [升級至 MDX 2.0](https://github.com/facebook/docusaurus/issues/4029)

## 結論

對 Docusaurus 來說，這是一個令人興奮的時刻。

我們受到關於 Docusaurus 的[正面反饋](https://x.com/sebastienlorber/timelines/1392048416872706049)的啟發，並且每天都會在網路上發現新的網站。

我們對這次的 Beta 版本感到非常興奮。我們在追求品質和穩定性的同時，也持續努力增加 Docusaurus 的採用率。對於那些還在猶豫是否要將現有的 Docusaurus 網站遷移到 Docusaurus 2 的使用者來說，現在是一個升級的好時機。我們希望在 Beta 階段結束時，您能運行在最新的基礎設施上。請告訴我們我們能如何提供協助。

感謝大家的閱讀，也感謝支持 Docusaurus 的整個社群。🤗
