## 重點功能

### Google Tag Manager 插件

Google Tag Manager 是一個提供高度靈活性的標籤管理系統，可讓分析與行銷團隊輕鬆載入其他第三方追蹤器並觸發分析標籤。

我們現在提供 [`@docusaurus/plugin-google-tag-manager`](/docs/api/plugins/@docusaurus/plugin-google-tag-manager/) 套件，您可以與現有的 [gtag.js 插件](/docs/api/plugins/@docusaurus/plugin-google-gtag/)搭配使用或替代之（請參閱[此文件](https://support.google.com/tagmanager/answer/7582054)了解何時該使用哪種解決方案）。

:::warning[Google 即將終止 Universal Analytics]

[Google 將於 **2023 年 7 月 1 日**終止其 Universal Analytics](https://blog.google/products/marketingplatform/analytics/prepare-for-future-with-google-analytics-4/)，並要求用戶遷移至 **Google Analytics 4**。

因此，我們也**棄用現有的 [`@docusaurus/plugin-google-analytics`](/docs/api/plugins/@docusaurus/plugin-google-analytics)** 套件。Docusaurus 用戶應建立新的 Google Analytics 4 資源，並遷移至 [gtag.js 插件](/docs/api/plugins/@docusaurus/plugin-google-gtag/)或 [Google Tag Manager 插件](/docs/api/plugins/@docusaurus/plugin-google-tag-manager/)。詳情與問題請參閱[專屬的 Docusaurus 議題](https://github.com/facebook/docusaurus/issues/7221)。

:::

### 標籤頁查詢字串支援

現在可以將選定的標籤頁連結至查詢字串參數。當選取標籤頁時，會以 `?qs-param=tab-value` 搜尋參數的形式儲存在瀏覽器網址中。

此功能允許深層連結至特定文件標籤頁，該標籤頁將在頁面載入時預先選取。

與 `groupId` 屬性並用時，查詢字串標籤值會優先於 `localStorage` 值。

請務必查看[文件](/docs/markdown-features/tabs#query-string)與下方示範以了解運作方式：

import BrowserWindow from '@site/src/components/BrowserWindow';
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

```tsx
<Tabs groupId="current-os" queryString>
  <TabItem value="android" label="Android">
    Android
  </TabItem>
  <TabItem value="ios" label="iOS">
    iOS
  </TabItem>
</Tabs>
```

<BrowserWindow>
  <Tabs groupId="current-os" queryString>
    <TabItem value="android" label="Android">
      Android
    </TabItem>
    <TabItem value="ios" label="iOS">
      iOS
    </TabItem>
  </Tabs>
</BrowserWindow>

### 巢狀警示框

現在可以透過為外層/包覆的警示框添加額外冒號，將一個警示框嵌套在另一個警示框內：

```md
::::tip[nested admonitions]

You can now nest one admonition within another!

:::danger

Use this sparingly when it makes sense.

:::

::::
```

::::tip[巢狀警示框]

您現在可以將一個警示框嵌套在另一個警示框內！

:::danger

請謹慎使用此功能，僅在有意義時採用。

:::

::::

### 部落格 `createFeedItems`

新的部落格插件選項 [`feedOptions.createFeedItems`](/docs/api/plugins/@docusaurus/plugin-content-blog/#CreateFeedItemsFn) 讓您能更靈活控制 RSS/Atom 饋送生成。現在可透過新的回調函式來轉換/過濾/限制饋送項目。

```tsx title="docusaurus.config.js"
[
  '@docusaurus/preset-classic',
  {
    blog: {
      feedOptions: {
        // highlight-start
        createFeedItems: async (params) => {
          const {blogPosts, defaultCreateFeedItems, ...rest} = params;
          return defaultCreateFeedItems({
            // keep only the 10 most recent blog posts in the feed
            blogPosts: blogPosts.filter((item, index) => index < 10),
            ...rest,
          });
        },
        // highlight-end
      },
    },
  },
];
```
