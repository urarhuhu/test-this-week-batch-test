## 重點更新

### 側邊欄項目描述

在 [#8236](https://github.com/facebook/docusaurus/pull/8236) 中，我們新增了為 `link` 和 `category` 類型的文檔側邊欄項目提供 `description` 屬性的功能。

```tsx title="sidebars.js"
[
  {
    type: 'link',
    label: 'Link with description',
    href: 'https://docusaurus.io',
    // highlight-next-line
    description: 'Some link description',
  },
  {
    type: 'category',
    label: 'Category with description',
    // highlight-next-line
    description: 'Some category description',
    items: [],
  },
];
```

這些描述將用於分類生成的索引頁面中。

![顯示帶有自訂描述的側邊欄分類生成索引頁面](./img/sidebar-item-description.jpg)

### 主題查詢字串

在 [#8708](https://github.com/facebook/docusaurus/pull/8708) 中，我們新增了透過 `docusaurus-theme` 查詢字串參數強制 Docusaurus 以 `light` 或 `dark` 模式初始化的功能。

這對於在 iframe 或 WebView 中嵌入現有 Docusaurus 頁面時確保主題一致性非常有用。

### 支援多個 gtag 追蹤 ID

在 [#8620](https://github.com/facebook/docusaurus/pull/8620) 中，我們為 [@docusaurus/plugin-google-gtag](/docs/api/plugins/@docusaurus/plugin-google-gtag) 插件新增了宣告多個追蹤 ID 的功能。

```js title="docusaurus.config.js"
module.exports = {
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        gtag: {
          trackingID: [
            // highlight-next-line
            'G-<YOUR-NEW-GA4-ID>',
            // highlight-next-line
            'UA-<YOUR-OLD-UA-ID>',
          ],
        },
      },
    ],
  ],
};
```

:::warning[Google 即將淘汰 Universal Analytics]

**[Google 將於 2023 年 7 月 1 日淘汰 Universal Analytics](https://blog.google/products/marketingplatform/analytics/prepare-for-future-with-google-analytics-4/)**。

Docusaurus 用戶應遷移至 Google Analytics 4。Google **不允許將現有 Universal Analytics 數據遷移**到新的 Google Analytics 4 資源中。

為保持分析數據的連續性，我們暫時建議您同時向兩個追蹤 ID 報告事件：舊的 (`UA-*`) 和新的 (`G-*`)。詳情請參閱**[專屬議題](https://github.com/facebook/docusaurus/issues/7221)**。

:::

### 翻譯功能

我們新增了以下元素的翻譯支援：

- [#8677](https://github.com/facebook/docusaurus/pull/8677) 引入新的 `process.env.DOCUSAURUS_CURRENT_LOCALE` (實驗性功能)，可讓您本地化設定檔，包括網站標題、標語、公告欄、baseUrl...
- [#8616](https://github.com/facebook/docusaurus/pull/8616) 允許翻譯導覽列和頁尾標誌的替代文字

我們新增了多種語言的預設主題翻譯支援：

- 🇭🇺 [#8668](https://github.com/facebook/docusaurus/pull/8668): 匈牙利文
- 🇳🇴 [#8631](https://github.com/facebook/docusaurus/pull/8631): 挪威文 (博克馬爾語)

:::tip

完成主題翻譯是一項[持續進行的工作](https://github.com/facebook/docusaurus/issues/3526)，也是為 Docusaurus 貢獻的簡單方式。我們會定期新增主題功能，通常需要[新的翻譯](https://github.com/facebook/docusaurus/issues/3526)。

:::

- [#8615](https://github.com/facebook/docusaurus/pull/8615): 修正使用深色導覽列時的顏色模式切換問題
- [#8699](https://github.com/facebook/docusaurus/pull/8699): 修正導覽列下拉選單的標籤焦點錯誤
