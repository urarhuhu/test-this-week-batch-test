# 📦 plugin-ideal-image

import APITable from '@site/src/components/APITable';

Docusaurus 插件，用於生成近乎理想的圖片（響應式、懶加載和低質量佔位圖）。

:::info

預設情況下，該插件在開發環境中處於**非活動狀態**，以便您始終可以查看全尺寸圖片。如果您想調試理想圖片行為，可以將 [`disableInDev`](#disableInDev) 選項設置為 `false`。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-ideal-image
```

## 使用方法 {#usage}

此插件僅支援 PNG 和 JPG 格式。

```jsx
import Image from '@theme/IdealImage';
import thumbnail from './path/to/img.png';

// your React code
<Image img={thumbnail} />

// or
<Image img={require('./path/to/img.png')} />
```

:::warning

此插件註冊了一個 [Webpack 加載器](https://webpack.js.org/loaders/)，會改變導入/請求的圖片類型：

- 之前：`string`
- 之後：`{preSrc: string, src: import("@theme/IdealImage").SrcImage}`

:::

:::warning[針對 pnpm 用戶]

從 [pnpm 10](https://github.com/pnpm/pnpm/releases/tag/v10.0.0) 開始，運行 `pnpm install` 預設不會執行依賴項的安裝腳本。為了讓我們的 `sharp` 圖片調整大小依賴項正確安裝，您需要額外的 pnpm 配置（[問題](https://github.com/lovell/sharp/issues/4343)），例如：

```json title="package.json"
{
  "pnpm": {
    "onlyBuiltDependencies": ["fsevents"]
  }
}
```

## 配置 {#configuration}

接受的字段：

```mdx-code-block
<APITable>
```

| 選項 | 類型 | 預設值 | 描述 |
| --- | --- | --- | --- |
| `name` | `string` | `ideal-img/[name].[hash:hex:7].[width].[ext]` | 輸出文件的文件名模板。 |
| `sizes` | `number[]` | _原始尺寸_ | 指定您想要使用的所有寬度。如果指定的大小超過原始圖片的寬度，則使用後者（即圖片不會被放大）。 |
| `size` | `number` | _原始尺寸_ | 指定您想要使用的一個寬度；如果指定的大小超過原始圖片的寬度，則使用後者（即圖片不會被放大） |
| `min` | `number` |  | 作為手動指定 `sizes` 的替代方案，您可以指定 `min`、`max` 和 `steps`，系統將為您生成尺寸。 |
| `max` | `number` |  | 參見上面的 `min` |
| `steps` | `number` | `4` | 配置在 `min` 和 `max` 之間（包括兩端）生成的圖片數量 |
| `quality` | `number` | `85` | JPEG 壓縮質量 |
| `disableInDev` | `boolean` | `true` | 您可以通過將此選項設置為 `false` 來在開發模式下測試理想圖片行為。**提示**：在瀏覽器中使用[網絡節流](https://www.browserstack.com/guide/how-to-perform-network-throttling-in-chrome)來模擬慢速網絡。 |

### 示例配置 {#ex-config}

以下是一個示例配置：

```js title="docusaurus.config.js"
export default {
  plugins: [
    [
      '@docusaurus/plugin-ideal-image',
      // highlight-start
      {
        quality: 70,
        max: 1030, // 調整後圖片的最大尺寸。
        min: 640, // 調整後圖片的最小尺寸。如果原始尺寸更小，則使用原始尺寸。
        steps: 2, // 在 min 和 max 之間（包括兩端）生成的圖片最大數量
        disableInDev: false,
      },
      // highlight-end
    ],
  ],
};
```
