# 📦 theme-search-algolia

此主題提供一個可輕鬆與 Algolia DocSearch 整合的 `@theme/SearchBar` 元件。與 `@docusaurus/theme-classic` 搭配使用時，能實現極簡的搜尋功能整合。更多資訊請參閱[搜尋功能](../../search.mdx)文件。

```bash npm2yarn
npm install --save @docusaurus/theme-search-algolia
```

此主題還新增了位於 `/search` 路徑的搜尋頁面（可替換的 `SearchPage` 元件），並支援 OpenSearch。您可透過 `themeConfig.algolia.searchPagePath` 變更預設路徑，設為 `false` 則停用搜尋頁面。

:::tip

若已安裝 `@docusaurus/preset-classic`，則無需另行安裝此主題作為依賴項。

:::
