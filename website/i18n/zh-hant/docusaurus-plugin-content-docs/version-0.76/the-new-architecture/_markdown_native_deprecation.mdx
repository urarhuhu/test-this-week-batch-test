:::info
原生模組（Native Module）與原生元件（Native Components）是現有架構中成熟的技術。
當新架構趨於穩定後，這些技術將逐步淘汰。新架構改採 [Turbo 原生模組](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/turbo-modules.md) 和 [Fabric 原生元件](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/fabric-native-components.md) 來實現相同效果。
:::
