title: Rect Object Type
---

`Rect` 接受數值型像素值來描述矩形區域的擴展距離。這些值會與原始區域尺寸相加以進行擴展。

## 範例

```js
{
    bottom: 20,
    left: null,
    right: undefined,
    top: 50
}
```

## 鍵與值

### `bottom`

| Type                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `left`

| Type                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `right`

| Type                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |

### `top`

| Type                        | Required |
| --------------------------- | -------- |
| number, `null`, `undefined` | No       |
