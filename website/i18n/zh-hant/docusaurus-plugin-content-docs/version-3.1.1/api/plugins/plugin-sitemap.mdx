# 📦 外掛程式-sitemap

import APITable from '@site/src/components/APITable';

此外掛程式會為您的網站建立站點地圖，以便搜尋引擎爬蟲能更準確地爬取您的網站。

:::warning[僅限生產環境]

此外掛程式在開發環境中始終處於非活動狀態，**僅在生產環境中啟用**，因為它作用於建置輸出。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-sitemap
```

:::tip

如果您使用預設配置 `@docusaurus/preset-classic`，則無需將此外掛程式安裝為依賴項。

您可以透過[預設選項](../../using-plugins.mdx#docusauruspreset-classic)來配置此外掛程式。

:::

## 配置 {#configuration}

接受的欄位：

```mdx-code-block
<APITable>
```

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `changefreq` | `string` | `'weekly'` | See [sitemap docs](https://www.sitemaps.org/protocol.html#xmlTagDefinitions) |
| `priority` | `number` | `0.5` | See [sitemap docs](https://www.sitemaps.org/protocol.html#xmlTagDefinitions) |
| `ignorePatterns` | `string[]` | `[]` | A list of glob patterns; matching route paths will be filtered from the sitemap. Note that you may need to include the base URL in here. |
| `filename` | `string` | `sitemap.xml` | The path to the created sitemap file, relative to the output directory. Useful if you have two plugin instances outputting two files. |

```mdx-code-block
</APITable>
```

:::info

此外掛程式也遵循部分網站配置：

- [`noIndex`](../docusaurus.config.js.mdx#noIndex)：會導致不生成站點地圖
- [`trailingSlash`](../docusaurus.config.js.mdx#trailingSlash)：決定站點地圖中的URL是否包含尾部斜線

:::

### 配置範例 {#ex-config}

您可以透過預設選項或外掛程式選項來配置此外掛程式。

:::tip

大多數Docusaurus使用者透過預設選項來配置此外掛程式。

:::

```js config-tabs
// Preset Options: sitemap
// Plugin Options: @docusaurus/plugin-sitemap

const config = {
  changefreq: 'weekly',
  priority: 0.5,
  ignorePatterns: ['/tags/**'],
  filename: 'sitemap.xml',
};
```

您可以在 `/sitemap.xml` 找到您的站點地圖。
