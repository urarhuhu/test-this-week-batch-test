# 📦 plugin-debug

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```

除錯外掛程式會在 [`http://localhost:3000/__docusaurus/debug`](http://localhost:3000/__docusaurus/debug) 顯示實用的除錯資訊。

這對外掛程式開發者特別有用，他們可以更輕鬆地檢查 `.docusaurus` 資料夾的內容（例如建立的路由），也能檢查從未寫入磁碟的資料結構，像是透過 `contentLoaded` 生命週期載入的外掛程式資料。

:::info

若透過經典預設集使用此外掛程式，預設情況下預設集會**在開發環境啟用外掛程式，並在生產環境停用** (`debug: undefined`)，以避免暴露潛在的敏感資訊。您可以使用 `debug: true` 來始終啟用，或 `debug: false` 來始終停用。

若使用獨立外掛程式，可能需要透過檢查環境來達到相同效果：

```js title="docusaurus.config.js"
export default {
  plugins: [
    // highlight-next-line
    process.env.NODE_ENV !== 'production' && '@docusaurus/plugin-debug',
  ].filter(Boolean),
};
```

:::

:::note

若您回報錯誤，我們可能會要求您在生產環境啟用此外掛程式，以便更輕鬆地檢查您的部署設定。

若您沒有任何敏感資訊，可以像我們一樣在生產環境保持啟用 [/__docusaurus/debug](/__docusaurus/debug)。

:::

## 安裝 {#installation}

```bash npm2yarn
npm install --save @docusaurus/plugin-debug
```

:::tip

若使用 `@docusaurus/preset-classic` 預設集，您無需將此外掛程式安裝為依賴項。

您可以透過[預設集選項](../../using-plugins.mdx#docusauruspreset-classic)來設定此外掛程式。

:::

## 設定 {#configuration}

此外掛程式目前沒有任何選項。

### 設定範例 {#ex-config}

您可以透過預設集選項或外掛程式選項來設定此外掛程式。

:::tip

大多數 Docusaurus 使用者透過預設集選項來設定此外掛程式。

:::

```mdx-code-block
<Tabs groupId="api-config-ex">
<TabItem value="preset" label="Preset options">
```

若使用預設集，請透過[預設集選項](../../using-plugins.mdx#docusauruspreset-classic)設定此外掛程式：

```js title="docusaurus.config.js"
export default {
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        // highlight-next-line
        debug: true, // This will enable the plugin in production
      },
    ],
  ],
};
```

```mdx-code-block
</TabItem>
<TabItem value="plugin" label="Plugin Options">
```

若使用獨立外掛程式，請直接提供選項給外掛程式：

```js title="docusaurus.config.js"
export default {
  // highlight-next-line
  plugins: ['@docusaurus/plugin-debug'],
};
```
