原生模組 (Native Module) 和原生元件 (Native Components) 是我們在舊架構中使用的穩定技術。
當新架構趨於穩定後，這些技術將被棄用。新架構使用 [Turbo 原生模組](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/turbo-modules.md) 和 [Fabric 原生元件](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/fabric-native-components.md) 來實現相似的功能。
:::
