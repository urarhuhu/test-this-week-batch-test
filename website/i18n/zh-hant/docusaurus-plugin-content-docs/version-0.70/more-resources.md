## 可嘗試的平台

[Expo](https://docs.expo.dev/) 是一套專為 React Native 打造的工具與服務框架，聚焦於協助您建構、發布與迭代應用程式。它提供類似網頁開發的預覽部署流程，並能自動化開發工作流程。Expo 甚至讓您能在不接觸 Xcode 或 Android Studio 的情況下開發 React Native 應用，同時也完全兼容這些工具的進階使用。

[Ignite](https://github.com/infinitered/ignite) 是一個內含多種 React Native 樣板的 CLI 起始工具包。最新版本 [Ignite Bowser](https://github.com/infinitered/ignite-bowser) 採用 MobX-State-Tree 狀態管理方案，整合 React Navigation 及其他常用函式庫。它提供元件、模型等生成器，並原生支援 Expo。若您正在尋找預配置的技術堆疊，Ignite 會是理想選擇。

## 尋找、製作並分享您的原生元件與 TurboModules

React Native 擁有由數千名開發者組成的社群，持續產出教學內容、工具、指南——以及原生元件！

在核心元件中找不到所需功能？造訪 [React Native 元件目錄](https://reactnative.directory)，探索社群開發的解決方案。

有興趣開發自己的原生元件或模組嗎？為特定使用情境製作模組並透過 NPM 和 GitHub 分享，能有效促進 React Native 生態系成長！請參閱開發指南：[原生模組（Android](native-modules-android.md)、[iOS](native-modules-ios.md)）與[原生元件（Android](native-components-android.md)、[iOS](native-components-ios.md)）。
