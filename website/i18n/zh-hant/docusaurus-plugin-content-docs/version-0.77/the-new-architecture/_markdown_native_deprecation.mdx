:::info
原生模組（Native Module）與原生元件（Native Components）是舊版架構採用的穩定技術。
當新版架構趨於穩定後，這些技術將被逐步淘汰。新版架構改採用 [Turbo 原生模組](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/turbo-modules.md) 與 [Fabric 原生元件](https://github.com/reactwg/react-native-new-architecture/blob/main/docs/fabric-native-components.md) 來實現相同效果。
:::
